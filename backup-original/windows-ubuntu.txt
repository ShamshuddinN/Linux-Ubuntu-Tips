1. Disable Bitlocker Encryption in windows
- Open Windows Powershell in Administrator Mode
- Check Encryption
>> manage-bde -status
- Disable Encryption:
i) Command Format: Disable-BitLocker -MountPoint <Drive Letter you want to decrypt>
>> Disable-BitLocker -MountPoint "C:"
